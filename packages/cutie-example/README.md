# @openstax/cutie-example

Example application demonstrating QTI v3 assessment items using the Cutie libraries.

## Development

```bash
# Start development server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview
```

## CI Scripts

```bash
# Run all CI checks
yarn ci

# Individual checks
yarn ci:lint
yarn ci:typecheck
yarn ci:spelling
```
